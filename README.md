# Python Scripts Collection

This repository contains small utility scripts I frequently used in my daily life, organized by folder.

## Core idea

Most scripts here are **self-bootstrapping**.

In most cases, you only need:
1. Install Python.
2. Open a terminal in this repository.
3. Run the script:

```powershell
py script-name.py
```

4. Follow the script's instruction(s)

On some systems, `python script-name.py` also works.

## Important note

Some scripts require extra setup, input files, or a specific run order.

If a script needs additional steps, go into that script folder and read its markdown guide.

## Recommended usage pattern

1. Pick a folder based on your task.
2. Open that folder.
3. Read the local `.md` guide first.
4. Run the script as documented.
